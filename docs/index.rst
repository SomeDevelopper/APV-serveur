# API AMZ

L'API AMZ est une interface de programmation permettant d'interagir avec la base de données de Amazon Prime Video, une société ou sont répertoriés les informations concernant tous les films, les acteurs et les réalisateur.

**Ici je tiens à souligner que cette base de données est avant tout fictive, car les données ont été généré par un script que j'ai conçu moi-même.**

Pour notre API, nous avons utilisé :

- le patron de conception DAO
- le langage python
- un serveur API développer en Flask
- une base de données postgreSQL sur supabase
- connecté la bd supabase sur le client DBeaver
- Et enfin nous avons généré une documentation grace à https://readthedocs.org/

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
